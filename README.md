# Bike Sales Data Analysis

## Project Overview

This project analyzes a bike sales dataset to understand customer behavior and identify key factors that influence bike purchases.
The dataset includes demographic and lifestyle information such as income, age, occupation, and commute distance, which are used to uncover patterns in purchasing decisions.


## Dataset Description

The dataset contains the following features:
* **ID** – Unique customer identifier
* **Marital Status** – Married (M) or Single (S)
* **Gender** – Male or Female
* **Income** – Annual income of the customer
* **Children** – Number of children
* **Education** – Education level
* **Occupation** – Customer's job role
* **Home Owner** – Whether the customer owns a house
* **Cars** – Number of cars owned
* **Commute Distance** – Distance traveled to work
* **Region** – Geographic region
* **Age** – Customer age
* **Purchased Bike** – Target variable (Yes/No)


## Objectives

* Analyze customer demographics and purchasing patterns
* Identify factors that influence bike purchases
* Perform exploratory data analysis (EDA)
* Generate insights for business decision-making


## Tools & Technologies

* Excel 


## Key Analysis Performed

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Relationship analysis between income, age, and bike purchases
* Region-wise and occupation-wise insights
* Visualization of trends and patterns


## Key Insights 

* Customers with higher income are more likely to purchase bikes
* Middle-aged individuals show higher purchase rates
* Commute distance impacts purchasing decisions
* Regional differences affect sales
